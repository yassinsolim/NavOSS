# Support

NavOSS is currently a Calgary technical alpha.

- Usage questions and reproducible bugs: [GitHub Issues](https://github.com/yassinsolim/NavOSS/issues)
- Security vulnerabilities: [private vulnerability report](https://github.com/yassinsolim/NavOSS/security/advisories/new)
- Beta limitations and useful report details: [docs/support.md](docs/support.md)
- Release readiness: [docs/release/testflight.md](docs/release/testflight.md)

Do not publish private addresses, precise trip histories, credentials, or sensitive screenshots in an issue. NavOSS support is not an emergency service.

The planned public support address is `navoss@yassin.app`. It must not be used until Cloudflare Email Routing is enabled and a delivery test passes.
